Testing OAuth Popup Flow:
  1. Open the main application: http://localhost:3001/en
  2. Look for the OAuth Debug Panel at the bottom of the page
  3. Click the "Sign in with Google" button in the header
  4. Watch the debug panel for real-time updates:
    • isInPopup - Shows if you're in a popup context
    • hasOAuthParams - Shows if OAuth parameters are detected
    • popupActive - Shows if popup OAuth is active
  5. Complete the Google OAuth flow
  6. The popup should close automatically and return you to the main page
Expected Flow:
  1. Click "Sign in with Google" ‚Üí Popup opens
  2. Complete Google authentication
  3. Google redirects to /auth/callback?code=...
  4. Our handler detects popup context and redirects to /auth/callback-popup
  5. Popup sends message to parent window with auth result
  6. Popup closes automatically
  7. Parent window receives auth result and completes login
Debugging Tips:
  • Open browser DevTools Console before starting
  • Look for console logs starting with [Auth] or [OAuthPopupHandler]
  • Check the Network tab to see OAuth redirects
  • If popup doesn't close, check for COOP/CORS errors in console
